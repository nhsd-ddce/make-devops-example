# Make DevOps Example

## Projects

- Java (Maven)
- React (JavaScript)

## Features

### Development

- Java (Maven)
  - Visual Studio Code project setup
  - Code autocompletion and formatting
  - Unit test execution from IDE
  - Reloading on change
  - Debugging
- React (JavaScript)
  - Reloading on change (non-Dockerised)
  - Debugging (in Firefox)
